# Brew Haven Cafe ☕

A modern and responsive café landing page built using **React, Tailwind CSS, and Framer Motion**.
The project showcases a stylish coffee shop website with smooth animations, an interactive menu, and a clean layout designed to provide a pleasant user experience.

## ✨ Features

* Responsive **modern landing page layout**
* Full-screen **hero section with background image**
* Attractive **About section with café story and highlights**
* Interactive **menu with images**
* **View More / Show Less** functionality for menu items
* **Customer testimonials** with profile images
* Smooth animations using **Framer Motion**
* Clean **footer with contact information**
